## 🎨 Приложение (Baby Diary)

### Функциональность
✅ **Мультипользовательское** - каждый пользователь = свой UUID
✅ **4 типа событий** - Сон 😴, Пробуждение 🌅, Кормление 🍼, Покупки 🛍️
✅ **Система заметок** - текстовые заметки с датой/временем
✅ **Аналитика** - 7-дневные графики сна/бодрствования
✅ **История** - полный список всех событий
✅ **Управление аккаунтом** - копирование ID, экспорт/импорт

### Технологии
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (0 зависимостей)
- **Backend**: Python 3.11, SimpleHTTPRequestHandler, JSON
- **Графики**: Chart.js (из CDN)
- **Хранение**: JSON файлы на сервере (data/{uuid}/diary.json)

### Вкладки приложения
1. **Запись** - быстрая запись событий + заметка
2. **Заметки** - просмотр и добавление текстовых заметок
3. **История** - полная история всех событий
4. **Анализ** - графики и статистика за неделю
5. **Настройки** - управление аккаунтом, импорт/экспорт

## 🐳 Docker

### Dockerfile
- Базовый образ: `python:3.11-slim` (~150MB)
- Копирует все файлы приложения
- Создает папку `/app/data` для данных
- Expose порт 8000

### docker-compose.yml
```yaml
- Сервис: baby-diary
- Порт: 8000:8000
- Том: diary-data для сохранения данных
- Restart: unless-stopped
- Health check: HTTP GET /
```

### Преимущества Docker
✅ Одна команда для установки: `docker-compose up -d`
✅ Полная изоляция от системы
✅ Легкое обновление и откат
✅ Совместимо с TrueNAS, Synology, и т.д.

